# World OF Wanos - Universo Web

Bienvenido al portal oficial de la hermandad **World OF Wanos** (UltimoWoW). Este es un ecosistema místico diseñado para la gloria, el orden y la purificación de Azeroth bajo la visión del Panteón.

## 🔗 Nexo de Almas
- **Discord Oficial:** [Únete aquí](https://discord.gg/s5AUDDmbta) - El centro de coordinación y hermandad.
- **Crónicas en Vivo (Twitch):** [Tilteados Anónimos TV](https://www.twitch.tv/tilteadosanonimostv) - El canal del frenesí de guerra.

## 🌌 Características Diamond-Tier
- **Lore Profundo:** Historia integrada de **Bhalanar** (Vigilante de la Urdimbre) y **Morgoroth** (Ejecutor de la Justicia Glacial).
- **Fase de Frenesí:** Sistema reactivo ("Modo Tilteados Anónimos") que transforma la atmósfera visual del sitio en tiempo real.
- **Ventanales del Vacío:** Integración de streaming en vivo para seguir las crónicas de la hermandad.
- **Reloj de la Eternidad:** Monitoreo preciso de las próximas incursiones a la Ciudadela de la Corona de Hielo.
- **Sabiduría y Forja:** Centro de conocimiento para el equipamiento óptimo y builds competitivas.

## 🚀 Implementación
Para desplegar este portal:
1. Sube los archivos a **Netlify**, **Vercel** o cualquier servidor estático.
2. Nota: El reproductor de Twitch detecta automáticamente el dominio del host para el parámetro de seguridad `parent`.

## 🛠️ Stack Técnico
- **Core:** HTML5 Semántico.
- **Visuals:** CSS3 avanzado con efectos de Glassmorphism y post-procesado cinemático.
- **Logic:** Vanilla JavaScript para el sistema de partículas y el estado de frenesí por eventos.
