# Contributing to World OF Wanos Web

Queremos que este sitio sea un reflejo de nuestra comunidad. Si deseas colaborar, sigue estas pautas:

## Cómo Colaborar
1.  **Reportar Errores**: Abre un Issue si encuentras algo que no funciona.
2.  **Sugerir Funciones**: Si tienes una idea épica, compártela en Discord.
3.  **Pull Requests**:
    *   Mantén el código limpio y comentado.
    *   Sigue el estilo de diseño Dark/Glacial establecido en `index.css`.
    *   No rompas el sistema de partículas.

## Estándares de Código
- Usa nombres de variables descriptivos en español o inglés técnico.
- Evita el uso de librerías externas si se puede resolver con JS Vainilla.
- La estética es prioridad absoluta.

¡Gracias por ayudar a expandir el Universo Wanos!
