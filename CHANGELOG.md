# Changelog

Todas las actualizaciones importantes del Universo Wanos se registrarán aquí.

## [1.0.0] - 2026-04-16
### Añadido
- Lanzamiento inicial de la web World OF Wanos.
- Secciones completas: Hero, Arquitectos, Pilares, Reglas, Reclutamiento.
- Sistema de diseño Glacial/Dark con variables CSS.
- Cronómetro de Raid dinámico (DK Style).
- Sistema de partículas de nieve y clima de Rasganorte.
- Forja de builds con consejos de clase.
- Galería y Panteón de rangos.
- Documentación institucional siguiendo el protocolo DarckRovert.
